# 🎂 Birthday Surprise Website

A romantic birthday surprise website built with HTML, CSS, and JavaScript.

## ✨ Features

* Animated birthday welcome page
* Interactive gift box experience
* Personal love letter section
* Video message gallery
* Background music support
* Mobile-friendly design
* Beautiful animations and effects

## 📁 Project Structure

```text
project/
│
├── index.html
├── giftbox.html
├── love-messages.html
├── video-letters.html
│
├── images/
├── videos/
└── music/
```
u need to create 3 folders in the seperately:
## 🖼️ Add Your Photos

Place your photos inside the `images` folder.

Example:

```text
images/
├── photo1.jpg
├── photo2.jpg
├── photo3.jpg
```
photo folder:
            *photo1
            *photo2
            .......
            *photo8(all photo u need to keep ur loved once)
Update the image paths in the HTML/JavaScript files if needed.

## 🎥 Add Your Videos

Place your videos inside the `videos` folder.

Example:

```text
videos/
├── video1.mp4
├── video2.mp4
```
videos folder:
            *video1
            *video2
            *video3
            *video4

Update the video paths in `video-letters.html`.

## 🎵 Add Your Music

Place your music files inside the `music` folder.

Example:

```text
music/
├── birthday-song.mp3
├── background-music.mp3
```
music folder:
            *firework1(which will be night shots sound for blast)
            *romantic-bg(for love letter bg music for 1st page)
            *romantic-bg1(starting page birthday song)
            *romantic-bg2(after romantic-bg completed this continuous)
            *song1
            *song2
            ......
            *song8(this all songs should be in 10sec for the photo frame bg song)

Update the audio source paths in the HTML files.

## 📱 Mobile Support

The website is designed to work on both desktop and mobile browsers.

## ❤️ Personalization

Replace:

* Photos in the `images` folder
* Videos in the `videos` folder
* Songs in the `music` folder
* Names, messages, and quotes inside the HTML files

to create a personalized birthday experience.
